# 💰 Expense Tracker

A simple command-line Expense Tracker built using Python. This project helps users record and manage daily expenses while practicing core Python concepts.

## ✨ Features

- Add new expenses
- View all expenses
- Calculate total expenses
- Menu-driven interface
- Simple and beginner-friendly

## 🛠️ Technologies Used

- Python

## 📚 Python Concepts Used

- Variables
- Lists
- Dictionaries
- While Loop
- For Loop
- If-elif-else
- Functions
- input()
- print()

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/tanishapatra90/Expense-Tracker.git
```

2. Open the project folder:

```bash
cd Expense-Tracker
```

3. Run the program:

```bash
python main.py
```
